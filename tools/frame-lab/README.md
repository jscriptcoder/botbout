# Frame Lab

Interactive tuner for frame data vs perception latency (L). Drag L and watch
which moves become reaction-blockable / whiff-punishable, and read the resulting
meta. Use it to pick frame numbers before running bot-vs-bot balance passes.

`frame-lab.jsx` is a single-file React component (Tailwind utility classes,
lucide-free). Drop it into a Vite/React app or render it in any React host.
